# 🎬 Movie Guide

Movie Guide is a React-based web application that helps users explore **Popular**, **Top Rated**, and **Upcoming** movies with quick access to IMDb pages for more details.  
It uses **The Movie Database (TMDb) API** to fetch real-time movie data and provides an intuitive UI for browsing and discovering films.

---

## 🚀 Features

- **Popular Movies** – See what's trending right now.
- **Top Rated Movies** – Discover the highest-rated films.
- **Upcoming Movies** – Stay updated on future releases.
- **Movie Details Page** – View:
  - 🎥 Movie poster image
  - ⭐ Ratings
  - 📝 Overview
  - 📊 Review count
  - 🎭 Genres
- **IMDb Page Redirection** – “Read More” button to open IMDb.
- **Responsive Design** – Works seamlessly across devices.
- **Dynamic Page Titles** – Updates tab title based on the current view.
- **Reusable Components** – Organized structure for scalability.
- **Smooth Navigation** – Handled with `useNavigate` from React Router.

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Routing:** React Router (with `useNavigate`)
- **Hooks:** `useState`, `useEffect`, custom `useFetch`
- **API:** [TMDb API](https://www.themoviedb.org/documentation/api)
- **Styling:** CSS, Bootstrap 5
- **Data Handling:** Fetch API + state management via hooks
